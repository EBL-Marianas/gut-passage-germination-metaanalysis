# Meta-analysis for the impacts of gut passage on germination for fleshy-fuited plants

This repository contains data and code to reproduce analyses presented in the mansucript "Frugivore gut passage increases seed germination: an updated meta-analysis" by Rogers et al. 2021. The 'clean' dataset include the experiments analyzed and presented in the manuscript. The 'raw' data include results of studies that did not fit the conditions for inclusion and more detailed notes and other metadata that may be relevant for future researchers.
